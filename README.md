# Javascript SDK for RIGO

`rigo-sdk-js` is Javascript SDK for RIGO blockhcain network.

## Usages
### Browserify

```bash
git clone https://github.com/rigochain/rigo-sdk-js.git
cd rigo-sdk-js
npm install
gulp
```

```html
<script type='text/javascript' src='./dist/rigo.min.js'></script>
```

### NodeJS

```bash
npm i rigo-sdk-js
```
