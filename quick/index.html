Quick Start

Prerequisite

$ node -v
v19.5.0

$ npm -v
9.4.1

$ protoc --version
libprotoc 3.21.4

Note

protoc 가 없을 경우, brew install protobuf 로 protobuf 최신 버전을 설치한다.

Browserify

git clone https://github.com/rigochain/rigo-sdk-js.git
cd rigo-sdk-js
npm install
gulp

<script type='text/javascript' src='./dist/rigo.min.js'></script>

NodeJS

npm i rigo-sdk-js
